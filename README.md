# 桌面便签 (Desktop Sticky Note)

一款轻量级桌面便签工具，支持置顶显示、悬浮球隐藏、一键恢复。

## 功能特性

- 📝 **便签编辑** - 富文本编辑，支持加粗、斜体、下划线、列表
- 📌 **置顶显示** - 便签窗口始终位于其他窗口之上
- 🔵 **悬浮球模式** - 收起便签为可拖动的悬浮球，节省屏幕空间
- 💾 **自动保存** - 内容实时自动保存，不怕丢失
- 🎨 **美观界面** - 经典便签黄色主题，简洁优雅
- 🖥️ **系统托盘** - 最小化到托盘，随时唤出

## 快速开始

### 环境要求

- Node.js >= 16
- npm >= 7

### 安装依赖

```bash
npm install
```

### 启动应用

```bash
npm start
```

### 打包构建

```bash
npm run build
```

## 使用说明

### 基本操作

1. **编辑便签** - 直接在编辑区输入内容，支持富文本格式
2. **置顶/取消置顶** - 点击标题栏的📌图标切换
3. **收起为悬浮球** - 点击标题栏的⊖图标，便签收起为屏幕上的小圆球
4. **恢复便签** - 单击悬浮球即可恢复便签窗口
5. **拖动悬浮球** - 按住悬浮球拖动到屏幕任意位置

### 快捷键

- `Ctrl+S` - 手动保存
- `Ctrl+B` - 加粗
- `Ctrl+I` - 斜体
- `Ctrl+U` - 下划线

## 技术栈

- **Electron** - 跨平台桌面应用框架
- **HTML/CSS/JS** - 前端渲染
- **electron-store** - 数据持久化
